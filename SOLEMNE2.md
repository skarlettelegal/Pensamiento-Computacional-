# Descripción Objetiva

## ¿Qué es el proyecto?

El proyecto es una imagen al estilo collage con manos que sobreponen una silueta de una mujer. Las interacciones buscan visibilizar las problemáticas de identidad que se han generado en la sociedad gracias a las redes sociales, ya que por ahí se ha establecido un estándar de belleza o de conductas que son impuestas sobre las mujeres y que se buscan cumplir dentro de la sociedad, o no solo cumplir, sino que también presionan a ser una persona que no eres.


## ¿Qué se ve en pantalla?

En pantalla se observa una composición digital basada en la imagen de una figura femenina como imagen principal. Sobre ella aparecen diferentes elementos visuales como manos en distintas posiciones, figuras geométricas (círculos, cuadrados y triángulos) y un cursor personalizado en forma de mano que sigue el movimiento del mouse.

También se muestran textos repetidos que representan frases asociadas a la presión estética y social, como:

- “Perfección”
- “Debes bajar de peso”
- “Eso no te queda”

Estos textos se desplazan y generan una sensación de saturación visual.

Al interactuar con el mouse, la pantalla cambia mediante diferentes estados:

- Fondos oscuros con transparencia
- Cambios de color aleatorios
- Mensajes como:
  - “Nunca seré suficiente”
  - “¿Tanto cuesta ser feliz?”

Esto genera una experiencia visual dinámica e interactiva.



## ¿Qué elementos visuales aparecen?

En el proyecto aparecen:

- Imágenes de una silueta de una mujer
- Manos en distintas posiciones
- Figuras geométricas:
  - Círculos
  - Cuadrados
  - Triángulos

Además, se integra texto repetitivo con frases asociadas a la presión estética y social.

La interacción del usuario modifica la visualidad mediante el mouse, generando:

- Cambios de color
- Opacidad
- Aparición de elementos gráficos e imágenes
- Transparencias
- Movimiento

Todo esto refuerza una sensación de saturación y tensión visual.



# Descripción Conceptual

## Idea central del proyecto y su relación con el sistema diseñado

La idea central es la presión por alcanzar un ideal femenino en las redes sociales y en la sociedad. Esto se relaciona con el sistema porque mientras más interacción existe con el cuerpo o con ciertos elementos visuales, la experiencia se vuelve más tensa e incómoda.

Al interactuar con el corazón o con elementos relacionados con la identidad femenina, se genera una tensión entre la mente y la reflexión personal.



## ¿Cuál es la regla de oro del sistema?

La regla de oro del sistema es:

> Mientras mayor sea la interacción, más dudas e inseguridades se generan, pero también existe la posibilidad de generar ayuda o reflexión.



## ¿Cómo se relaciona esta lógica con la problemática de género elegida?

La lógica del proyecto se relaciona con la problemática de los ideales de belleza y la presión social impuesta sobre las mujeres.

El sistema interactivo simula cómo una persona es constantemente:

- Observada
- Evaluada
- Corregida

Esto se representa mediante la interacción del mouse y distintas condiciones visuales que alteran la escena.


# Input / Output y Sistema

## ¿Qué datos entran? (INPUT)

Estos inputs permiten que el usuario active distintas zonas de la obra al moverse por el lienzo.

Los datos que ingresan al sistema son:

- Posición del mouse (`mouseX`, `mouseY`)
- Estado del mouse (`mouseIsPressed`)
- Movimiento en el tiempo (`frameCount`)
- Aleatoriedad generada por `random()`



## ¿Cómo se procesan y transforman?

El sistema procesa estos datos mediante condiciones y funciones:

- Las estructuras `if / else if / else` determinan qué escena se muestra según la posición del mouse.
- `random()` introduce variación en:
- Los bucles `for` repiten textos e imágenes para crear saturación visual.
  
## ¿Qué respuesta visual producen? (OUTPUT)

Como resultado, el sistema genera distintas respuestas visuales dinámicas:

- Aparición de textos repetitivos asociados a presión estética y social.
- Cambio de fondos y capas oscuras con transparencia.
- Activación de imágenes como manos y un corazón según la interacción.
- Efectos de movimiento, rotación y aleatoriedad en elementos gráficos.
- Mensajes finales como:
  - “Nunca seré suficiente”
  - “¿Tanto cuesta ser feliz?”

Todo esto refuerza la carga conceptual

# Pensamiento Computacional

## Reglas que gobiernan el sistema

La interactividad del proyecto se basa en la relación entre el movimiento del usuario y la respuesta del sistema visual.

El usuario no controla directamente una narrativa fija, sino que activa diferentes estados emocionales del sistema al moverse por zonas específicas del lienzo.

Esto genera una experiencia:

- Fragmentada
- No lineal

La interacción simula:

- Presión
- Vigilancia
- Saturación visual

Conectando directamente con la problemática de los estándares de belleza y la exigencia social en redes.

```javascript
// VARIABLES
let cursorMano; // variable donde guardo la imagen del cursor personalizado (mano)
let rotacion = 2; // variable numérica que uso para generar rotación constante en los elementos de la mente
let corazon; // variable donde se almacenará la imagen del corazón cargada en preload
let intensidad = 1; // Variable para alteral la intensidad de un texto, esto altera su transparencia

function preload() {
  // Funcion para cargar las imagenes en el lienzo
  imagen1 = loadImage("Mujer.jpg"); // cargo la imagen de fondo principal antes de iniciar el sketch
  imagen2 = loadImage("Mano1.png"); // cargo primera imagen de mano semicerrada
  imagen3 = loadImage("mano2.png"); // cargo segunda imagen de mano abierta
  imagen4 = loadImage("manoarriba.png"); // cargo imagen de mano semicerrada con angulo
  imagen5 = loadImage("corazon2.jpg"); // cargo imagen que representa el corazon que se activa en interacción
  cursorMano = loadImage("ManoCursor.png"); // cargo imagen que reemplaza el cursor tradicional, que es una mano indicando
}

function setup() {
  createCanvas(400, 600); // creo un lienzo de 400x600 píxeles
  noCursor(); // oculto el cursor normal para usar el personalizado
  rectMode(CENTER); // cambio el modo de referencia del rectángulo al centro en vez de la esquina
}

// función separada para dibujar el cursor personalizado
function cursorPersonalizado() {
  image(cursorMano, mouseX, mouseY, 90, 90); // dibuja la mano siguiendo la posición del mouse en tiempo real
}

function draw() {
  fill(0, intensidad);
  rotacion = rotacion + 0.6; // aumento constantemente la variable rotación para generar movimiento continuo

  // MUNDO VISUAL DE LA MUJER
  image(imagen1, 0, 0, width, height); // coloco la imagen de la mujer ocupando todo el lienzo
  noStroke(); // elimino los bordes de figuras

  fill(247, 200, 211); // asigno color rosado para las figuras
  square(300, 350, 80); // dibujo un cuadrado en zona derecha inferior
  square(50, 180, 90); // dibujo un segundo cuadrado en zona izquierda

  fill(168, 181, 138); // asigno color a verde  para contraste visual
  circle(350, 300, 100); // dibujo un círculo en el lado derecho del lienzo
  circle(80, 250, 90); // dibujo un círculo en el lado izquierdo del lienzo

  fill(247, 200, 211); // asigno el color rosado nuevamente
  circle(60, 290, 50); // dibujo un círculo pequeño al lado inferior izquierdo del lienzo
  circle(370, 260, 50); // dibujo un círculo pequeño al lado inferior derecho del lienzo

  // ELEMENTOS HOSTILES

  image(cursorMano, mouseX, mouseY, 90, 90); // dibujo el cursor personalizado en la posición del mouse, como un elemento
  fill(0); //asigno el color negro para representar tensión
  triangle(0, 0, 0, 50, 40, 60); // dibujo un triángulo superior izquierdo como elemento agresivo visual en el lienzo
  triangle(400, 400, 400, 350, 360, 380); // dibujo un triángulo inferior derecho como elemento de presión visual en el lienzo
  triangle(400, 0, 370, 70, 370, 0); // dibujo un triángulo superior derecho en el lienzo

  image(imagen2, 250, 100, 150, 150); // carga la imagen de la mano en la parte superior derecha del lienzo
  image(imagen3, 250, 400, 150, 150); // carga la imagen de la mano en la parte inferior derecha del lienzo
  image(imagen4, 20, 0, 150, 150); //carga la imagen de la mano en la esquina superior izquierda del lienzo

  push(); // inicio push para guardar la siguente configuracion y que se repita
  translate(350, 300); // translado el punto de origen de la imagen a x,y
  rotate(90); // hago la rotación de la imagen en grados
  image(imagen2, 250, 100, 150, 150); // se ve la imagen de la mano en la parte inferior izquierda del lienzo
  pop(); // termino aislamiento para no afectar otros elementos

  push(); // inicio push para guardar la siguente configuracion y que se repita
  translate(60, 900); // translado la imagen de la mano para el lado izquierdo del lienzo
  rotate(10); // hago una rotación para que se vea la mano en el lienzo
  image(imagen3, 250, 400, 150, 150); // se ve la imagen de la mano la cual fue cambiada de lugar
  pop(); // cierro la configuracion anterior

  // INTERACCIÓN en el cuerpo
  // condición para detectar una zona especifica con mouse presionado, que indicva la presión en el cuerpo de la mujer y como afecta dentro de ella
  if (
    mouseX > 130 &&
    mouseX < 200 &&
    mouseY > 30 &&
    mouseY < 500 &&
    mouseIsPressed
  ) {
    fill(0); // fondo negro que representa saturación
    square(200, 200, 800); // dibujo un cuadrado para cubrir todo el lienzo para generar el cambio de estado

    // FIGURAS EN MOVIMIENTO dentro del cambio de estado

    push(); // inicio push para la siguente configuracion del rectángulo
    fill(59, 59, 59); // asigno el color gris oscuro para sensación de peso visual
    translate(100, 150); // posiciono el rectangulo desde su centro
    scale(3); // agrando figura para impacto visual
    rotate(rotacion + 4); // asigno rotación dinámica basada en la variable
    rect(0, 0, 50, 150); // dibujo un rectángulo en el lienzo
    pop(); // cierro la configuracion del rectángulo

    push(); // inicio push para la siguente configuracion para el cudrado
    fill(105, 105, 105); // asigno el gris más claro para contraste
    translate(300, 570); // posiciono en parte inferior del lienzo
    scale(2); // escalo el cuadrado a 2
    rotate(rotacion + 3); // rotación constante con la variable
    square(50, 10, 120); // dibujo un cuadrado en el lienzo
    pop(); // cierro la configuracion del rectángulo del cuadrado

    // TEXTO
    fill(255, 0, 0); // asigno el color rojo para representar la presión
    textSize(30); // asigno el tamaño del texto, que es grande para generar impacto visual, sigue teniendo el color negro de la primera condición

    for (let x = 0; x < 100; x++) {
      // bucle que repite texto para simular saturación

      text("Eso no te queda", x * 220 - frameCount * 10, 50);
      text("Haz esto", x * 160 - frameCount * 7, 100);
      text("Perfección", x * 150 - frameCount * 12, 200);
      text("Debes casarte", x * 250 - frameCount * 12, 250);
      text("Belleza", x * 100 - frameCount * 10, 300);
      text("Debes bajar de peso", x * 300 - frameCount * 10, 350);
      text("Come esto", x * 160 - frameCount * 9, 400);
      text("Ser responsable", x * 250 - frameCount * 12, 450);
      text("Te verías bien si...", x * 260 - frameCount * 10, 500);
    }
  }

  // INTERACCIÓN  2 en la cabeza
  else if (mouseX > 130 && mouseX < 200 && mouseY > 30 && mouseY < 100) {
    // condición 2, si el mouse pasa por la cabeza de la mujer, pasa lo siguente:
    textAlign(CENTER, CENTER); // centro el texto para que sea más legible
    square(200, 200, 800); // dibujo un cuadrado que sera el fondo
    fill(255); // asigno el color blanco para contraste

    for (let x = 0; x < width; x = x + 35) {
      // creo un for para circulos en la parte de X del lienzo
      for (let y = 0; y < height; y = y + 25) {
        // creo un for para circulos en la parte de Y del lienzo
        // doble bucle para crear patrón repetido en toda la pantalla

        fill(random(260), 50, 50); // color aleatorio para generar inestabilidad visua, que va entre el rojo y el gris
        ellipse(x, y, random(20)); // dibujo círculos con el tamaño variable en random para generar el efecto caótico
        textSize(15); // asigno el tamaño de texto
        text("Nunca seré suficiente", 200, 300); // escribe el mensaje central del estado emocional de la mujer luego o durante la presion
      }
    }
  } else {
    // si ninguna de las dos condicones anteriores se cumple, pasara lo siguente:
    noStroke(); // elimino los bordes
    fill(0, 70); //asigno el color negro con transparencia para efecto
    square(width / 2, height / 2, max(width, height) * 2); // dibujo un cuadrado que cubre toda la pantalla generando sensación de vacio o incomodidad
  }

  // INTERACCIÓN de la mujer

  intensidad = random(50, 255); // Asigno un valor a la variable creada al principio, este sera aleatorio entre los valores 50 y 255 del rgb

  if (mouseX > 0 && mouseX < 80 && mouseY > 200 && mouseY < 250) {
    // creo una condicional que si el mouse pasa por la zona indicada entre x, y pasara lo siguente:
    fill(247, 200, 211); // asigno el fondo de color rosado
    square(0, 0, 1200); // dibujo un cuadrado que cubre lienzo completo

    fill(255, 255, 255, intensidad); //asigno el color blanco para el texto con el cambio de intensidad
    textSize(20); //asigno el tamaño del texto
    textAlign(CENTER, CENTER); // asigno centrar el texto
    text("¿Quién soy?", 200, 200); //escribo la pregunta identitaria que tiene un efecto de intensidad, ya que aun es confuso en su mundo
  }

  // INTERACCIÓN
  else if (
    // si la priemra condición no se cumple, se activa la siguente condición, en la parte asignada del lienzo
    mouseX > 0 &&
    mouseX < 70 &&
    mouseY > 170 &&
    mouseY < 350 &&
    mouseIsPressed
  ) {
    fill(247, 200, 211); // asigno el mismo color de fondo para coherencia visual
    square(0, 0, 1200); //dibujo un cuadrado para cubrir todo el lienzo
    textAlign(CENTER, CENTER); // centro texto
    fill(random(255), random(255), random(255)); // asigno un color aleatorio para sensación emocional inestable
    text("¿Que tengo que hacer?", 200, 200); //escribo el texto de presión social
  }

  // INTERACCIÓN CORAZÓN

  let sobreCorazon = // defino zona del corazón como un cuadrado invisible
    mouseX > 150 && mouseX < 180 && mouseY > 100 && mouseY < 150;

  if (sobreCorazon && mouseIsPressed) {
    // creo la condicion si el mouse se presiona sobre la zona indicada anteriormente pasara lo siguente:
    image(imagen5, 0, 0, width, height); //asigno una nueva imagen como respuesta emocional, que aun puede encontrarse
    fill(255); //asigno el color blanco para legibilidad y representar paz
    textSize(24); // asigno el tamaño del texto
    textAlign(CENTER, CENTER); // centro el texto
    text("¿Tanto cuesta ser feliz?", width / 2, height / 2);
    // escribo el mensaje final del sistema interactivo como reflexion
  }
}
```
