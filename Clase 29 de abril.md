## Repaso: Algoritmos

Un algoritmo es una secuencia de instrucciones ordenadas y finitas que permiten resolver un problema o realizar una tarea.

### Características
- Preciso: cada paso debe ser claro.
- Ordenado: sigue una secuencia lógica.
- Finito: debe terminar.
- Definido: con los mismos datos entrega el mismo resultado.

### Estructura de un algoritmo
- Input → datos de entrada.
- Proceso → transformación de los datos.
- Output → resultado final.

## Diagramas de flujo

Representación visual de un algoritmo.

Sirven para:
- Planificar programas antes de escribir código.
- Visualizar decisiones y procesos.
- Comunicar la lógica de un sistema.

Utilizan símbolos estándar para que cualquier persona pueda interpretarlos.

Se puede imaginar como una serie de compartimentos numerados donde cada posición guarda un valor.
